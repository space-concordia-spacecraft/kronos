uint32_t kronos::FileOpenMessage::mode

The mode used to open the file. The modes include: KS_OPEN_MODE_READ_ONLY - Reading only. KS_OPEN_MODE_WRITE_ONLY - Writing only. KS_OPEN_MODE_WRITE_READ - Reading and Writing. KS_OPEN_MODE_APPEND - Sets file offset to end-of-file prior to each write. KS_OPEN_MODE_CREATE - Creates file if it doesnt exist. KS_OPEN_MODE_EXCL - Throws an error if the file already exists. must be combined with KS_OPEN_MODE_CREATE. KS_OPEN_MODE_TRUNCATE - Truncate the opened file to size zero.
